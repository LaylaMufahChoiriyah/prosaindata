Langkah-langkah

  • Buka ElephantSQL kemudian login dengan menggunakan akun yang sudah terdaftar

  • Pada bagian Details klik Browser kemudian masukkan perintah sql untuk create tabel kemudian klik execute

CREATE TABLE Data_Iris (
    ID  SERIAL PRIMARY KEY,
    SepalLength NUMERIC(3,1) NOT NULL,
    SepalWidth  NUMERIC(3,1) NOT NULL,
    PetalLength NUMERIC(3,1) NOT NULL,
    PetalWidth  NUMERIC(3,1) NOT NULL,
    Variety     VARCHAR(10) NOT NULL
);
